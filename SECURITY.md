# Security Policy

## Supported Versions

We currently provide security updates for the following:

| Component | Supported |
| --------- | --------- |
| Latest released version of each NuGet package | :white_check_mark: |
| AssistStudio desktop app: latest published Microsoft Store build | :white_check_mark: |
| Older versions | :x: |

Please ensure you are on the most recent release to receive security patches.

## Reporting a Vulnerability

We take the security of AssistStudio seriously. If you believe you have found a security vulnerability, please follow these steps:

1. **GitHub Private Security Advisory (Priority 1):** Please use the [GitHub Private Security Advisory](https://github.com/fieldcure/FieldCure-AssistStudio/security/advisories/new) feature to report the vulnerability privately. This is our preferred method of communication as it allows for secure collaboration and coordinated disclosure.

2. **Private Email:** If you are unable to use the GitHub advisory feature, please report the vulnerability by emailing **jhkim@fieldcure.com**.

Please do not open a public issue for security vulnerabilities. We will investigate all reports and strive to provide a resolution as quickly as possible.
