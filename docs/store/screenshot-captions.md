Run open-weight models locally with Ollama. Fully offline after setup — no API key, no network call, no data leaves your machine.

## Tips

- Partner Center allows a separate caption per language (English / Korean) for each screenshot.
- Order screenshots **01 → 08** in the listing — the order is preserved in the Store.
- The hero shot (01) appears first; pick what best demonstrates the marquee feature.
